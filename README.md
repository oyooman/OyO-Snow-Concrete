## OyO-Snow-Concrete

- Snow concrete is a mod that adds new Hazard and tinted variaton of concrete and refined concrete snow compatible for better IMMERSION.

## Hazard concrete
- adding : emergency-concrete, operations-concrete, radiation-concrete, warning-concrete, express-concrete, green-concrete, safety-concrete.
